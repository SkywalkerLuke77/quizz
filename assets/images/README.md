# Bilder für die Quizfragen

Lege hier eure eigenen Fotos ab, benannt passend zu den Pfaden in `js/questions.js`:

```
q01.jpg
q02.jpg
q03.jpg
...
q16.jpg
```

Empfohlenes Format: **JPG, 1600×1000 px (16:10), gut komprimiert (< 500 KB)**,
damit das Display (`display.html`) die Bilder schnell und scharf anzeigt.

Falls eine Datei fehlt, zeigt die Anzeige automatisch einen eleganten
Platzhalter mit dem Ringe-Motiv an – die App funktioniert also auch ganz
ohne eigene Fotos.
